## 📌 Overview

A production-ready gym website featuring all the sections a real fitness business needs — from a hero banner to membership pricing and a contact form. Built mobile-first with attention to performance and user experience.

---

## ✨ Features

- 🎨 **Hero Banner** — Full-viewport landing section with CTA button
- 💪 **Services Section** — Workout categories with icon cards
- 💳 **Membership Plans** — Pricing tiers with comparison layout
- 👨‍🏫 **Trainer Profiles** — Team section with cards
- 📬 **Contact Form** — JavaScript-validated form with real-time feedback
- 📱 **Fully Responsive** — Mobile-first design with breakpoints for all devices
- 🍔 **Hamburger Menu** — Animated mobile navigation menu
- 🔝 **Sticky Navbar** — Shrinks on scroll for clean UX
- ✨ **Smooth Scroll Animations** — Sections animate in as you scroll
- ⚡ **Performance Optimized** — Minimal CSS, no unused dependencies

## 📐 CSS Highlights

- **Mobile-first** media queries (`min-width` approach)
- CSS **Flexbox** for nav and card layouts
- CSS **Grid** for service and plan sections
- **CSS custom properties** (variables) for consistent theming
- **`@keyframes`** animations for section reveals
